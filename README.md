# RFT UART Interface Software

## Overview

The RFT UART Interface Software provides a reference implementation for interfacing with RFT Series Force/Torque Sensors via UART communication.
It enables real-time data acquisition, monitoring, and basic control of the sensor, and is intended for evaluation, integration, and development purposes.
This repository includes both executable files and full source code to support rapid deployment and customization in embedded and PC-based environments.

## Key Features
* Real-time force/torque data acquisition via UART
* Configurable communication parameters (COM port, baud rate)
* Sensor scaling via divider parameter settings
* Bias and un-bias control for sensor calibration
* Adjustable internal digital filter (cutoff frequency selection)
* Data logging for analysis and validation
* Example implementation for quick integration
 
## Note
This software is provided as a reference example. Users are encouraged to modify and extend it according to their specific application requirements.
